# QS University Rankings Dashboard Dataset

## Project Description
This project prepares a cleaned dataset for a QS University Rankings Dashboard.

The Python script merges the 2026 QS dataset with historical university rankings from 2023 and 2024.

## Files

- University_Raw_Data.csv
- 2023 QS World University Rankings.csv
- 2024 QS World University Rankings.csv
- University_Final_Data.csv
- merge.py

## Technologies Used

- Python
- Pandas
- CSV

## Output

The final dataset contains:
- 2026 Rank
- Previous Rank (2025)
- 2024 Rank
- 2023 Rank
- University information
- QS KPI metrics
