# EduVision – Higher Education Performance Dashboard

## Project Overview

EduVision is a higher education analytics project that integrates multiple global university ranking datasets to support comparative analysis of universities, research performance, and student metrics.

## Data Sources

- QS World University Rankings 2026
- Times Higher Education Rankings
- CWUR Rankings
- THE Student Dataset 2023

## Work Completed

### Data Cleaning
- Cleaned and standardized university ranking datasets
- Standardized university and country names

### Data Integration
- Performed exact matching across ranking systems
- Applied fuzzy matching to resolve university name variations
- Created a unified dataset of 480 universities

### Dataset Statistics
- QS Rankings
- THE Rankings
- CWUR Rankings
- THE 2023 Student Dataset
- 480 common universities identified and merged

## Files Included

- Cleaned datasets
- Fuzzy match details
- Master merged university dataset
- Data integration notebook

## Current Status

Data integration completed. Missing value handling, KPI engineering, and Tableau dashboard development are in progress.
