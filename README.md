# EduVision_DV – Higher Education Performance Dashboard

## Project Overview

EduVision_DV is a data visualization project that analyzes the performance of higher education institutions across the world. The project uses global university ranking datasets to create interactive Tableau dashboards that provide insights into university rankings, research performance, student diversity, and country-wise education trends.

---

## Objectives

- Collect and integrate global university datasets.
- Clean and preprocess educational data.
- Generate key performance indicators (KPIs).
- Build interactive Tableau dashboards.
- Compare universities and countries using educational metrics.

---

## Dataset

The project uses publicly available higher education datasets, including:

- QS World University Rankings
- Times Higher Education World University Rankings

Dataset files are stored in CSV format.
