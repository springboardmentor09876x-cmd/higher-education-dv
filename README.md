# Higher Education Performance Dashboard

## Infosys Springboard Internship Project

This repository contains my work for the Infosys Springboard Data Visualization Internship.

### Project Overview

The Higher Education Performance Dashboard analyzes global university data using QS World University Rankings and Times Higher Education (THE) World University Rankings. The project aims to build interactive Tableau dashboards that provide insights into university performance, research impact, student diversity, and country-wise education trends.

### Milestone 1 - Data Collection

- Collected QS World University Rankings dataset
- Collected THE World University Rankings dataset
- Integrated both datasets into a common structure
- Generated `university_raw_data.csv`

### Milestone 2 - Data Cleaning & Transformation

- Removed duplicate records
- Standardized university names
- Standardized country names
- Cleaned and transformed ranking metrics
- Handled missing values
- Converted columns to appropriate data types
- Prepared a Tableau-ready dataset
- Generated `university_cleaned.csv`
